Privacy Policy

Last updated: March 2026 · X-Brain Chrome Extension v3.8
Summary: X-Brain does not collect, store, or share any personal data with third parties. All data remains stored locally in your own browser only.

1. Data We Collect

X-Brain stores the following data exclusively in your browser's local storage. This data is never transmitted to any server:

  • Groq API key (stored locally in your browser only)
  • Tone, language, and autopilot preferences
  • Daily comment usage counter
  • Comment history (last 50 comments, local only)
  • VIP account list

2. Data We Do Not Collect

X-Brain does not collect, store, or transmit the following under any circumstances:

  • Personal identifiers (name, email address, physical address, etc.)
  • X/Twitter account credentials or passwords
  • Financial or payment information
  • Location data
  • Web browsing history
  • User activity tracking or analytics data

3. Groq API Usage

X-Brain uses Groq's AI API to generate comments. Tweet text is sent to Groq's servers solely for comment generation purposes. In this process:

  • Your Groq API key is stored only in your browser and never shared with the developer
  • Tweet content is processed temporarily and is subject to Groq's own privacy policy
  • The developer has no access to this API traffic at any point

You can review Groq's privacy policy at groq.com/privacy-policy.

4. Third-Party Sharing

Your data is never shared, sold, or rented to any third party. X-Brain does not use any analytics, advertising, or tracking services.

5. Data Security

All data is stored exclusively on your local device via Chrome's built-in chrome.storage.local API. The developer cannot access this data in any way.

6. Children's Privacy

X-Brain is intended for users aged 18 and older only. Use of this extension by individuals under 18 is not permitted.

7. Policy Changes

Any changes to this policy will be announced in the extension's update notes. The current policy will always be published on this page.

8. Contact

For privacy-related questions, please contact: furkan@xbrain.app
